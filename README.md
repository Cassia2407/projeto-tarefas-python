# Sistema de Gerenciamento de Tarefas

Este é um projeto simples desenvolvido em Python que permite gerenciar tarefas diretamente pelo terminal.

## Funcionalidades

* Adicionar tarefas
* Listar tarefas
* Remover tarefas
* Tratamento de erros (entradas inválidas)
* Interface via terminal

## Tecnologias utilizadas

* Python

## O que aprendi

* Estruturas de repetição (while)
* Condicionais (if/elif)
* Listas em Python
* Funções
* Tratamento de erros (try/except)
* Lógica de programação

## Como executar

1. Clone o repositório:
   git clone https://github.com/Cassia2407/projeto-tarefas-python.git

2. Entre na pasta:
   cd projeto-tarefas-python

3. Execute o programa:
   python tarefas.py
