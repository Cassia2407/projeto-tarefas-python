# Sistema de Gerenciamento de Tarefas

Projeto simples desenvolvido em Python para gerenciar tarefas no terminal.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Remover tarefas
- Salvamento automático em arquivo

## Tecnologias
- Python

## Como executar

1. Baixe o projeto
2. Execute no terminal:
python tarefas.py
